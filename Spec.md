## Spec

- Headers

  ```
    # H1
    # H2
    # H3
    # H4
    # H5

  ```

- Code block support using backtick

  ````
    ``` console.log("test)```
  ````

- Bold
  ```
    ** Hello **
  ```
- Italics
  ```
    * Helo *
  ```
- Auto link urls
- Auto link email as mailto links
- strike through
  ```
   ~~Hello~~
  ```
